# Advent of Code 2020
My submissions for Advent of Code 2020.

## Running
Node is required to run the solutions. I am personally using `yarn` but that is optional. To build the project run:

```shell
npm install
```

Then to compute the solution for one of the days, run:

```shell
npm run day <1-25>
```

Or if you want to run all of the days:

```shell
npm run all
```
